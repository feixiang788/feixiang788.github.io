* the idea that is represented by a word, phrase, etc
* the idea that a person wants to express by using words, signs, etc.

（3）第三种方法：Representing words by their context

思考：这个假设有个突出问题，前提是收集到的句子中词的用法不存在问题，并且存在同一个词在不同语境下含义不同，上下文也不同。
词向量：有上述假设，我们将为每个单词构造稠密词向量，词向量的选取将使得相邻的上下文词向量之间的相似性会较高

3 Word2Vec概述

Word2Vec(Mikolov et al. 2013) is a framework for learning word vector
Mikolov并不是第一个提出单词的分布式表示，在他之前Yoshua Bengio也曾提出一种相似的方法。
学习算法本身决定维度。
主要思想：
* 我们拥有大量文字语料
* 在固定字典中每个单词都用向量去表示
* 遍历文本中的每个位置t，在每个位置上有一个中心词c（center word），有一些上下文单词o（context word）。
* 使用词向量c和o之间的相似性去计算已知c的情形下o出现的概率
* 不断修正词向量从而达到概率最大值
一开始的词向量都是随机向量，然后再开始优化
